# Path of program

Path of program

## Usage

``` r
doc_path(filename = NULL, path = NULL)
```

## Arguments

- filename:

  Name of output file

- path:

  program path

## Value

character string

## Examples

``` r
# \donttest{
 doc_path(filename = "my_tbl.pdf", path = NULL)
#> [1] "/home/runner/work/_temp/759cf65e-5093-42bb-80b8-f02dc3e70b00"
# }
```
