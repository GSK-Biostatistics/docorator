# Path of program

Path of program

## Usage

``` r
doc_path(filename = NULL, path = NULL)
```

## Arguments

- filename:

  Name of output file

- path:

  program path

## Value

character string

## Examples

``` r
if (FALSE) { # \dontrun{
 doc_path(filename = "my_tbl.pdf", path = NULL)
} # }
```
