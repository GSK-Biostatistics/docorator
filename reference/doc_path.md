# Path of program

Path of program

## Usage

``` r
doc_path(filename = NULL, path = NULL)
```

## Arguments

- filename:

  Name of output file

- path:

  program path

## Value

character string

## Examples

``` r
# \donttest{
 doc_path(filename = "my_tbl.pdf", path = NULL)
#> [1] "/home/runner/work/_temp/8449bf6a-f138-4269-a6a4-3b47ba299a6d"
# }
```
