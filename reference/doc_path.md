# Path of program

Path of program

## Usage

``` r
doc_path(filename = NULL, path = NULL)
```

## Arguments

- filename:

  Name of output file

- path:

  program path

## Value

character string

## Examples

``` r
# \donttest{
 doc_path(filename = "my_tbl.pdf", path = NULL)
#> [1] "/home/runner/work/_temp/5b2bf8a8-6148-4ee4-aa65-2bff15129de9"
# }
```
