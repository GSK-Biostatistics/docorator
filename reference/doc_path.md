# Path of program

Path of program

## Usage

``` r
doc_path(filename = NULL, path = NULL)
```

## Arguments

- filename:

  Name of output file

- path:

  program path

## Value

character string

## Examples

``` r
# \donttest{
 doc_path(filename = "my_tbl.pdf", path = NULL)
#> [1] "/home/runner/work/_temp/a7d2c719-8c27-4f03-9e3f-3fd409626732"
# }
```
